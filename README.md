# 📅 Python Calendar

A simple Python application that allows users to:

- View the current month's calendar.
- Enter any valid date.
- Calculate the number of days remaining until that date.
- Calculate the number of days passed since a previous date.
- Handle invalid dates using exception handling.

## Technologies Used

- Python
- datetime
- calendar

## Features

- Displays today's date.
- Prints the current month's calendar.
- Validates user input.
- Calculates the difference between two dates.
- Displays the selected month's calendar.
